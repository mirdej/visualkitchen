Usage Tables

This file list the contents of the hi.tools usage tables database. Wherever a usage or usage page has to be specified, the symbol from either the Name or Alias column of the following tables can be given instead of the numeric values. These symbols are case-insensitive.

You may also modify the sqlite3 file "hi.usagetables.db" to define your own aliases (e. g. with the sqlite3 command-line utility). If you do so, take care to backup your modified database file before upgrading the hi.tools!

A python script to generate this file from the database file is included in the hi.tools source distribution.
